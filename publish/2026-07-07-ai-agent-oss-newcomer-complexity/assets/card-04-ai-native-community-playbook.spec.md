# Card 04 — AI Native 社区/团队落地手册

## Goal

Provide practical guidance for teams adopting AI coding agents without weakening newcomer learning or long-term maintainability.

## Recommended layout

Use a 2x2 playbook card:

### 1. 保留新人入口
- Keep starter tasks visible
- Pair new members with agent-assisted tasks
- Make first PR review educational, not just pass/fail

### 2. 管住复杂度
- CI complexity checks
- small PR boundaries
- refactoring budget
- owner review for sensitive areas

### 3. 记录协作过程
- AI-assisted disclosure
- task prompt / intent summary
- changed-files scope
- validation evidence

### 4. 用对照评估效果
- baseline period
- matched teams / projects
- compare trend, not anecdotes
- separate adoption proxy from usage intensity

## Main headline

**让 Agent 扩大人的能力，而不是吞掉人的成长路径**

## Visual details

- 2x2 rounded cards, each with a short title and 3–4 bullets.
- Use simple icons: doorway, wrench, ledger, chart.
- Keep it action-oriented and concise.
- Do not overstate the paper; frame as practical playbook derived from the findings.

## Insert location

After `给我们的启示`.
