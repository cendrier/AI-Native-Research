# Visual Card 03 — 12 次运行里的完成、差异和修正

## Purpose
Show the paper's quantitative findings without overclaiming.

## Layout
Use a four-metric dashboard card.

Metric 1: "12 / 12"
Label: "workspaces passed completion gate"
Note: "4 篇 scientific ML papers × 3 runs"

Metric 2: "158"
Label: "recorded targets matched with report coverage"
Note: "完成是 workspace state，不是 final message"

Metric 3: "37 / 39"
Label: "scalar anchor-run observations inside paper-anchored threshold"
Note: "matched ≠ exact reproduction"

Metric 4: "1.2h → 13.0h"
Label: "elapsed replication time range"
Note: "过程差异被 workspace records 保存"

Add a small correction-work strip:
"25 superseded tracked executions → 21 in the two PINN papers"

## Key message
"同样叫完成，背后的 fidelity、耗时、修正路径和判断规则可能完全不同。"

## Visual style
Use metric cards with short explanatory captions. Avoid celebratory language. Make the nuance visible: completion is constrained by recorded targets and validation rules.
