# Visual Card 02 — Agent Work / Workspace Records / Validation Checks

## Purpose
Turn the paper's Figure 1 into a reader-friendly explanatory card for AI Native Engineering teams.

## Layout
Three horizontal rows, aligned by columns.

Row 1 title: "Agent Work"
Columns:
1. Inspect materials
2. Record target list
3. Reconstruct method
4. Run experiments
5. Compare & report

Row 2 title: "Workspace Records"
Columns:
1. Source inventory + hashes
2. Reproduction matrix + task ledger
3. Specification files
4. Run record + provenance
5. Report coverage + replication report

Row 3 title: "Validation Checks"
Columns:
1. Paper-asset checks
2. Active-target check
3. Specification check
4. Provenance + comparison checks
5. Completion gate

## Annotation
Add a side note:
"关键不是让 agent 更会说，而是让每一步都留下可检查的文件。"

## Footer
"可复现工作 = 执行 + 记录 + 外部验证"

## Visual style
Use a clean workflow matrix. Keep English labels where they are paper terms, but add short Chinese descriptors if useful. Avoid copying the exact paper figure; create a simplified editorial card.
