# Visual Card 01 — 从最终回答到 Evidence Contract

## Purpose
Explain the core shift of the article: agent completion should be judged by auditable evidence, not by a confident final message.

## Layout
Use a horizontal before/after layout with a strong center arrow.

Left panel: "Prompt-only / Final Message"
- Agent says: "完成了"
- Chat transcript is the main record
- Hard to resume after interruption
- Hard to distinguish generated output from copied or substitute evidence
- Human reviewer sees conclusion but not proof path

Center arrow label:
"把交付物从 answer 变成 workspace"

Right panel: "Evidence Contract / Inspectable Workspace"
- Target list / reproduction matrix
- Task ledger / open questions
- Run records / commands / outputs
- Provenance / code / config / seed / source passages
- Comparison evidence / report coverage
- Completion gate outside the agent

## Key message
"AI Agent 的真正交付物，不是最终回答，而是可验证的证据链。"

## Visual style
Clean diagram, light background, rounded cards, minimal icons. Use one small badge: "Paper-replication". Keep text concise and readable.
