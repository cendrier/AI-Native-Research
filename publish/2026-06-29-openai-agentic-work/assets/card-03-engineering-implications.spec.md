Title:
AI Native Engineering Implications

Subtitle:
Agentic Work 对研发组织的 6 个启发

Six blocks:
1. Measure delegation, not usage
2. Design for verification
3. Build memory as infrastructure
4. Prepare for non-engineer technical execution
5. Manage parallel agent load
6. Turn workflows into skills

Short takeaway:
AI Native Engineering 的核心，不是“更多 AI 工具”，而是把 agentic work 变成可委派、可验证、可记忆、可复用的工作系统。
