# Visual Card Spec — card-02-social-adoption-map

## Title

Agent 采用是社会扩散

## Purpose

Show that the strongest initial-use signals in the Microsoft rollout were social-exposure signals, not demographics.

## Layout

Create a network-style card with one engineer node in the center and three surrounding exposure rings.

- Reviewer peers
  - Label: 经常互相 review 的同事
  - Data: 25%+ peers active → initial use odds +54%; retention +30%

- Skip-level peers
  - Label: 同一个 skip-level 组织里的同事
  - Data: 25%+ peers active → initial use odds +216%; retention +66%
  - Emphasize as strongest signal.

- Direct manager
  - Label: 直属 manager 可见使用
  - Data: manager active → initial use odds +82%; retention +22%

Bottom insight bar: “AI agent rollout 更像工程实践扩散，而不是 SaaS 开通。”

## Visual style

Use compact data callouts. Make the +216% number prominent but not sensational. The card should feel like an evidence map, not a marketing infographic.

## Footer attribution

Source: Microsoft / arXiv, 2026-07-01. Metrics are odds lifts vs. no coworker-use reference group.
