# Card 03 — 自给自足栈：从 observer 到 autonomous operator

## Goal

Show that the real output of an AI-native engagement is not only a deployed system, but a repeatable internal capability.

## Layout

Use a layered stack with a maturity ladder on the side.

Layer 1: Deployed systems
- Production agentic workflows
- Business process integration

Layer 2: Semantic layer
- Enterprise data sources
- Metadata enrichment
- Governed, versioned knowledge graph

Layer 3: Operating assets
- Runbooks
- Architecture docs
- Tests and evaluation
- Release and rollback paths

Layer 4: People and ownership
- Internal champions
- Customer engineers as co-builders
- Autonomous operators

Side ladder:
Observer → Co-builder → Autonomous operator

## Design notes

- Make it feel like a capability-building stack, not a vendor services diagram.
- Use short Chinese labels with one-line explanations.
