# Card 04 — 给我们的 AI Native Engineering Playbook

## Goal

Convert the AWS FDE signal into a practical internal playbook.

## Layout

Use a 2x2 or checklist card.

Section 1: 组织打法
- 建 AI enablement pod
- 短周期嵌入业务研发团队
- 绑定 business / engineering / security owner

Section 2: 交付产物
- Agent workflow
- Skills / prompts / automation scripts
- Tests, evaluation, runbook
- Permission and cost boundaries

Section 3: 基础设施
- Code and service semantic layer
- Architecture docs and ADRs
- Incident history and release rules

Section 4: 衡量指标
- Time to first production workflow
- Review pass rate
- Reusable workflow count
- Internal champion count
- Unit task cost and quality

## Design notes

- This card should be directly actionable for engineering leaders.
- Avoid generic inspiration language; emphasize what to build and measure.

## Footer

Internal synthesis based on AWS FDE announcement and AWS AI-DLC methodology.
