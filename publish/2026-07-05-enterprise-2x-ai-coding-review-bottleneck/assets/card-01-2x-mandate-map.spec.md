# Card 01 — 2× Mandate 不是开关，而是组织飞轮

## Purpose

Help readers understand that the enterprise 2× result is not a one-step tool deployment. It is a multi-stage organizational loop.

## Suggested layout

A left-to-right flywheel or pipeline with five nodes:

1. 明确 2× Mandate
2. 工具可用 + 真实任务入口
3. 持续使用与经验积累
4. PR throughput 上升
5. Review / validation 成为新瓶颈

Add a small warning callout under the final node:

> 2× 不是 license 激活率，而是整个工程系统被重新分配。

## Key data to show

- 802 developers
- 196,212 PRs
- Jan 2024 – Apr 2026
- 2.09× per-capita merged PR throughput by Apr 2026

## Visual style

Use the canonical AI Native Research visual style. Clean, high-contrast, minimal decoration. Emphasize the final bottleneck node visually, but do not use alarming colors unless allowed by the style file.
