# Card 02 — 从 1.5× 到 2×：积累使用释放收益

## Purpose

Show that productivity gains are gradual and tied to accumulated AI tool use, not an immediate post-announcement jump.

## Suggested layout

Use a simple curve or stepped timeline:

- Baseline: pre-mandate 1.0×
- Observed average use: ~1.5× within-developer throughput
- Sustained use around nine months: ~2× estimated throughput
- April 2026 organization-level result: 2.09× per-capita merged PR throughput

Add a small label near the curve:

> Mandate 是催化剂，持续使用才是机制。

## Key interpretation

Avoid implying hard causality. Use phrasing like “observed / estimated” rather than “guaranteed.”

## Visual style

Clean data card with one large curve, three annotated markers, and one short caveat box.
