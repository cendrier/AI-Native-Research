# AI Native Research

Building the AI Native Engineering Playbook.

本仓库用于沉淀 AI Native 组织、研发效能、agentic engineering、AI-native PDLC 和软件开发方法论的研究资产。目标不是收集文章，而是把高质量案例转化为可复用的 case study、framework 和 pattern。


## 工作原则

> Don't collect articles. Build reusable knowledge.

每天优先关注最近两周内发表、具备实际落地效果和方法论价值的文章、工程博客、案例研究或报告。只有真正值得长期保留的内容才进入仓库。
